Enlaces en HTML

Aprende a conectar páginas y recursos web mediante enlaces HTML

Los enlaces en HTML (<a>) son elementos fundamentales que permiten la navegación entre páginas y recursos. El atributo href especifica el destino del enlace.

Tipos de Enlaces

Enlaces Externos

Redirigen a otras páginas web fuera del sitio actual. Generalmente se abren en una nueva pestaña.

Enlaces Internos

Conectan con otras páginas dentro del mismo sitio web, manteniendo la navegación del usuario.

Enlaces Ancla

Llevan a una sección específica dentro de la misma página mediante identificadores.

Código Fuente

<!-- Enlace externo -->
<a href="https://www.google.com" target="_blank">
    Ir a Google
</a>

<!-- Enlace interno -->
<a href="estructura.html">
    Ir a la página de Estructura
</a>

<!-- Enlace ancla -->
<a href="#seccion-ejemplo">
    Ir a la sección de ejemplo
</a>
