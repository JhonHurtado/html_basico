Tipos de Inputs en HTML

Explora los diferentes tipos de campos de entrada para formularios HTML

Los elementos <input> son fundamentales en HTML para permitir la interacción del usuario con los formularios. Cada tipo de input tiene un propósito específico y se define mediante el atributo type.

Tipos Principales de Inputs

Texto

type="text" - Campo de entrada para texto libre.

Email

type="email" - Campo específico para correos electrónicos.

Contraseña

type="password" - Campo que oculta el texto ingresado.

Número

type="number" - Campo para valores numéricos.

Fecha

type="date" - Selector de fechas del calendario.

Archivo

type="file" - Campo para seleccionar archivos.

Código Fuente

<form>
    <div class="mb-3">
        <label for="nombre" class="form-label">Nombre:</label>
        <input 
            type="text" 
            class="form-control" 
            id="nombre" 
            placeholder="Escribe tu nombre"
        >
    </div>

    <div class="mb-3">
        <label for="email" class="form-label">Email:</label>
        <input 
            type="email" 
            class="form-control" 
            id="email" 
            placeholder="correo@ejemplo.com"
        >
    </div>

    <div class="form-check">
        <input 
            class="form-check-input" 
            type="radio" 
            name="genero" 
            id="masculino"
        >
        <label class="form-check-label" for="masculino">
            Masculino
        </label>
    </div>

    <button type="submit" class="btn btn-primary">
        Enviar
    </button>
</form>
