Abre la consola de tu navegador (F12 o Ctrl + Shift + J) para ver los resultados de tus ejercicios.

Variables

Las variables en JavaScript son contenedores que almacenan valores. Existen tres formas de declararlas:

  • var: Forma tradicional, con alcance de función.
  • let: Introducida en ES6, con alcance de bloque y puede ser reasignada.
  • const: También de ES6, con alcance de bloque pero no puede ser reasignada.

Ejemplo:

var edad = 30;          // Puede ser redeclarada y reasignada
let nombre = "Ana";     // Puede ser reasignada pero no redeclarada
const PI = 3.14159;     // No puede ser reasignada ni redeclarada

// Las variables pueden almacenar diferentes tipos de datos
let esEstudiante = true;   // Booleano
let calificacion = 9.5;    // Número decimal
let noDefinido;            // undefined (sin valor asignado)

Arreglos (Arrays)

Los arrays son colecciones ordenadas de elementos que pueden contener cualquier tipo de dato. Se crean usando corchetes [].

Métodos comunes de arrays:

  • push(): Añade elementos al final del array
  • pop(): Elimina y devuelve el último elemento
  • unshift(): Añade elementos al inicio del array
  • shift(): Elimina y devuelve el primer elemento
  • slice(): Extrae una porción del array sin modificarlo
  • splice(): Modifica el array (añadir/eliminar/reemplazar elementos)

Ejemplo:

// Creación de un array
let colores = ["rojo", "verde", "azul"];

// Acceso a elementos por índice (comienza en 0)
console.log(colores[0]);  // "rojo"

// Modificación de elementos
colores[1] = "amarillo";

// Añadir elementos
colores.push("morado");  // Añade al final
colores.unshift("negro"); // Añade al inicio

// Eliminar elementos
let ultimoColor = colores.pop();   // Elimina y devuelve "morado"
let primerColor = colores.shift(); // Elimina y devuelve "negro"

Objetos (Objects)

Los objetos en JavaScript son colecciones de pares clave-valor que pueden contener propiedades y métodos. Se crean usando llaves {}.

Características de los objetos:

  • Las propiedades pueden accederse con notación de punto (objeto.propiedad) o corchetes (objeto["propiedad"])
  • Pueden contener cualquier tipo de dato, incluyendo funciones (métodos)
  • Son mutables: se pueden añadir, modificar o eliminar propiedades después de su creación

Ejemplo:

// Creación de un objeto
let persona = {
  nombre: "Juan",
  edad: 25,
  esEstudiante: true,
  hobbies: ["música", "deportes", "lectura"],
  direccion: {
    calle: "Av. Principal",
    ciudad: "Madrid"
  },
  saludar: function() {
    return "Hola, soy " + this.nombre;
  }
};

// Acceso a propiedades
console.log(persona.nombre);        // "Juan"
console.log(persona["edad"]);       // 25
console.log(persona.hobbies[0]);    // "música"
console.log(persona.direccion.ciudad); // "Madrid"
console.log(persona.saludar());     // "Hola, soy Juan"

// Añadir o modificar propiedades
persona.telefono = "555-1234";
persona.edad = 26;

Condicionales

Las estructuras condicionales permiten ejecutar diferentes bloques de código dependiendo de si una condición es verdadera o falsa.

Tipos de estructuras condicionales:

  • if...else: Evalúa una condición y ejecuta diferentes bloques de código
  • switch: Evalúa una expresión y ejecuta código según el caso que coincida
  • Operador ternario: Forma abreviada de escribir condiciones simples (condición ? expr1 : expr2)

Ejemplo:

// if...else
let hora = 14;

if (hora < 12) {
  console.log("Buenos días");
} else if (hora < 18) {
  console.log("Buenas tardes");
} else {
  console.log("Buenas noches");
}

// switch
let diaSemana = 2;
let nombreDia;

switch (diaSemana) {
  case 1:
    nombreDia = "Lunes";
    break;
  case 2:
    nombreDia = "Martes";
    break;
  // ... casos para otros días ...
  default:
    nombreDia = "Día inválido";
}

// Operador ternario
let edad = 20;
let mensaje = edad >= 18 ? "Eres mayor de edad" : "Eres menor de edad";

Ciclos (Loops)

Los ciclos permiten ejecutar un bloque de código repetidamente mientras se cumpla una condición.

Tipos de ciclos:

  • for: Se usa cuando se conoce el número de iteraciones
  • while: Se ejecuta mientras una condición sea verdadera
  • do...while: Similar a while, pero garantiza al menos una ejecución
  • for...of: Itera sobre elementos de objetos iterables (arrays, strings)
  • for...in: Itera sobre las propiedades de un objeto

Ejemplo:

// for
for (let i = 0; i < 5; i++) {
  console.log(`Iteración ${i}`);
}

// while
let contador = 0;
while (contador < 3) {
  console.log(`Contador: ${contador}`);
  contador++;
}

// do...while
let j = 0;
do {
  console.log(`J vale: ${j}`);
  j++;
} while (j < 2);

// for...of (para iterables)
let frutas = ["manzana", "pera", "uva"];
for (let fruta of frutas) {
  console.log(fruta);
}

// for...in (para objetos)
let personaLoop = {nombre: "Ana", edad: 28};
for (let propiedad in personaLoop) {
  console.log(`${propiedad}: ${personaLoop[propiedad]}`);
}

Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica. Pueden recibir parámetros y devolver valores.

Formas de declarar funciones:

  • Declaración de función: function nombre() {}
  • Expresión de función: const nombre = function() {}
  • Función flecha (ES6): const nombre = () => {}

Ejemplo:


// Declaración de función
function sumar(a, b) {
  return a + b;
}

// Expresión de función
const restar = function(a, b) {
  return a - b;
};

// Función flecha
const multiplicar = (a, b) => a * b;

// Función con parámetros por defecto
function saludar(nombre = "Invitado") {
  return `Hola, ${nombre}!`;
}

// Llamar a las funciones
console.log(sumar(5, 3));        // 8
console.log(restar(10, 4));      // 6
console.log(multiplicar(2, 6));  // 12
console.log(saludar());          // "Hola, Invitado!"
console.log(saludar("María"));   // "Hola, María!"
