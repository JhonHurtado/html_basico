Elementos Semánticos Principales

Descubre los elementos que dan significado a tu estructura HTML

<header>
Encabezado principal

Define el encabezado de una página o sección, generalmente contiene navegación, logos y títulos principales.

<header>
  <h1>Mi Sitio</h1>
  <nav>...</nav>
</header>
<nav>
Navegación

Contiene enlaces de navegación principales del sitio web, facilitando la orientación del usuario.

<nav>
  <ul>
    <li><a href="#">Inicio</a></li>
  </ul>
</nav>
<main>
Contenido principal

Envuelve el contenido principal único de la página, excluyendo navegación, footers y sidebars.

<main>
  <article>...</article>
  <section>...</section>
</main>
<article>
Contenido independiente

Representa contenido independiente que puede ser distribuido o reutilizado por separado.

<article>
  <h2>Título del artículo</h2>
  <p>Contenido...</p>
</article>
<section>
Sección temática

Agrupa contenido relacionado temáticamente, generalmente con un encabezado identificativo.

<section>
  <h2>Servicios</h2>
  <p>Lista de servicios...</p>
</section>
<aside>
Contenido complementario

Contiene información relacionada pero secundaria al contenido principal.

<aside>
  <h3>Enlaces relacionados</h3>
  <ul>...</ul>
</aside>

Buenas Prácticas

Un solo <main> por página
Cada página debe tener únicamente un elemento main.
Jerarquía de encabezados
Mantén una estructura lógica h1 → h2 → h3.
Usa <article> para contenido independiente
Posts de blog, noticias, comentarios.
Agrupa con <section>
Para contenido temáticamente relacionado.

Beneficios del HTML Semántico

SEO Optimizado

Los motores de búsqueda comprenden mejor la estructura y jerarquía del contenido, mejorando el ranking.

Accesibilidad Mejorada

Los lectores de pantalla navegan más eficientemente, beneficiando a usuarios con discapacidades.

Código Mantenible

Estructura más clara y organizada que facilita el mantenimiento y desarrollo en equipo.

Mejor UX

Navegación más intuitiva y experiencia de usuario coherente en todos los dispositivos.

Compatibilidad Universal

Mejor soporte en diferentes dispositivos, navegadores y tecnologías asistivas.

Ejemplo Completo

Estructura semántica completa 
<!DOCTYPE html>
<html lang="es">
<head>
    <title>Mi Sitio Semántico</title>
    <meta name="description" content="Descripción">
</head>
<body>
    <header>
        <h1>Mi Sitio Web</h1>
        <nav>
            <ul>
                <li><a href="#inicio">Inicio</a></li>
                <li><a href="#acerca">Acerca</a></li>
                <li><a href="#contacto">Contacto</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <header>
                <h2>Título del Artículo</h2>
                <time datetime="2025-06-14">14 Junio, 2025</time>
            </header>
            <p>Contenido del artículo principal...</p>
            <footer>
                <p>Por: <span>Autor</span></p>
            </footer>
        </article>
        
        <section>
            <h2>Sección Relacionada</h2>
            <p>Información adicional temática...</p>
        </section>
    </main>
    
    <aside>
        <h3>Contenido Relacionado</h3>
        <nav>
            <h4>Enlaces Útiles</h4>
            <ul>
                <li><a href="#">Recurso 1</a></li>
                <li><a href="#">Recurso 2</a></li>
            </ul>
        </nav>
    </aside>
    
    <footer>
        <p>© 2025 Mi Sitio Web</p>
        <nav>
            <a href="#privacidad">Privacidad</a>
            <a href="#terminos">Términos</a>
        </nav>
    </footer>
</body>
</html>
