Tablas en HTML

Aprende a organizar datos de manera estructurada usando tablas HTML

Las tablas en HTML se utilizan para mostrar datos organizados en filas y columnas. Son una herramienta fundamental para presentar información estructurada de manera clara y accesible.

Elementos Básicos de una Tabla

<tr>

Define una fila en la tabla (table row). Actúa como contenedor para las celdas.

<th>

Define un encabezado de columna (table header). Se usa para identificar el contenido de las columnas.

<td>

Define una celda de datos (table data). Contiene la información real de la tabla.

Ejemplo Práctico

Inventario de Frutas

Producto Precio Cantidad
Manzana $1.00 5
Banana $0.50 10
Naranja $0.80 8

Código Fuente

<table class="table table-bordered table-hover">
    <thead class="table-primary">
        <tr>
            <th scope="col">Producto</th>
            <th scope="col">Precio</th>
            <th scope="col">Cantidad</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Manzana</td>
            <td>$1.00</td>
            <td>5</td>
        </tr>
        <tr>
            <td>Banana</td>
            <td>$0.50</td>
            <td>10</td>
        </tr>
        <tr>
            <td>Naranja</td>
            <td>$0.80</td>
            <td>8</td>
        </tr>
    </tbody>
</table>
