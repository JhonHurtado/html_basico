Estructura de un Documento HTML

Aprende los elementos fundamentales que componen una página web HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Todo archivo HTML tiene una estructura básica compuesta por etiquetas principales que definen su organización y contenido.

Estructura Básica

DOCTYPE

<!DOCTYPE html>

Define el tipo de documento como HTML5. Es la primera línea de todo documento HTML.

HTML

<html>

Es el elemento raíz que contiene todo el documento HTML. Define el inicio y fin del documento.

HEAD

<head>

Contiene metadatos, título de la página, enlaces a CSS y otros recursos no visibles.

BODY

<body>

Contiene todo el contenido visible de la página web como texto, imágenes y enlaces.

Ejemplo Práctico

Vista Previa

Hola, Mundo!

Este es el resultado visual del código HTML

Código Fuente

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>Hola, Mundo!</h1>
</body>
</html>
