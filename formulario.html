Formularios en HTML

Aprende a crear formularios interactivos para recopilar información de los usuarios

Los formularios en HTML son elementos interactivos que permiten recopilar información de los usuarios. Se crean con la etiqueta <form> y pueden contener diversos tipos de campos de entrada para capturar diferentes tipos de datos.

Elementos Básicos de un Formulario

Campos de Entrada

<input>

Define campos para diferentes tipos de datos: texto, email, número, contraseña, etc.

Área de Texto

<textarea>

Crea un campo para texto multilínea, ideal para comentarios o mensajes largos.

Listas Desplegables

<select>

Crea menús desplegables para seleccionar una o varias opciones.

Ejemplo Práctico

Formulario de Contacto

Código Fuente

<form action="#" method="POST">
    <div>
        <label for="nombre">Nombre:</label>
        <input type="text" 
               id="nombre" 
               name="nombre" 
               required>
    </div>

    <div>
        <label for="email">
            Correo Electrónico:
        </label>
        <input type="email" 
               id="email" 
               name="email" 
               required>
    </div>

    <div>
        <label for="mensaje">
            Mensaje:
        </label>
        <textarea id="mensaje" 
                  name="mensaje" 
                  rows="4">
        </textarea>
    </div>

    <button type="submit">
        Enviar
    </button>
    <button type="reset">
        Borrar
    </button>
</form>
