Iframes en HTML

Aprende a incorporar contenido externo en tus páginas web

La etiqueta <iframe> permite incrustar contenido web externo como mapas, videos o páginas completas dentro de tu sitio. Es una herramienta versátil para integrar contenido sin redirigir a los usuarios.

Atributos Comunes

src

Especifica la URL del contenido que se va a incrustar en el iframe.

width y height

Definen el ancho y alto del iframe en píxeles o porcentaje.

allowfullscreen

Permite que el contenido se pueda ver en pantalla completa.

Ejemplo de Iframe

Mapa Incrustado

Código del Iframe

<iframe 
    src="https://www.google.com/maps/embed?..."
    width="600" 
    height="450" 
    style="border:0;" 
    allowfullscreen="" 
    loading="lazy">
</iframe>
