Imágenes en HTML

Aprende a incorporar y optimizar imágenes en tus páginas web

Las imágenes en HTML se añaden utilizando la etiqueta <img>. Esta etiqueta no necesita una etiqueta de cierre y requiere al menos el atributo src para especificar la ubicación del archivo de la imagen.

Atributos Principales

src

Define la ruta de la imagen. Puede ser una ruta local o una URL externa.

alt

Texto alternativo que se muestra si la imagen no se carga. Importante para accesibilidad.

width/height

Especifican el ancho y alto de la imagen en píxeles.

Código Fuente

<!-- Imagen desde URL -->
<img 
    src="https://via.placeholder.com/300"
    alt="Imagen de ejemplo desde URL"
    class="img-fluid rounded"
>

<!-- Imagen local -->
<img 
    src="imagenes/ejemplo.jpg"
    alt="Imagen local"
    class="img-fluid rounded"
>
