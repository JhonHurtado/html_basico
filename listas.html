Listas en HTML

Aprende a organizar contenido de forma estructurada usando listas HTML

Las listas en HTML son elementos fundamentales que permiten organizar y estructurar contenido de manera clara y coherente. Existen diferentes tipos de listas para diferentes propósitos de organización.

Tipos de Listas

Lista Ordenada

<ol> - Los elementos se enumeran automáticamente.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Lista No Ordenada

<ul> - Los elementos se marcan con viñetas.

  • Elemento
  • Otro elemento
  • Último elemento

Ejemplos Prácticos

Receta de Cocina

Pasos a seguir:

  1. Preparar los ingredientes
  2. Mezclar los ingredientes
  3. Cocinar a fuego lento
  4. Servir y disfrutar

Ingredientes:

  • Harina
  • Azúcar
  • Huevos
  • Leche

Código Fuente

<!-- Lista Ordenada -->
<h4>Pasos a seguir:</h4>
<ol>
    <li>Preparar los ingredientes</li>
    <li>Mezclar los ingredientes</li>
    <li>Cocinar a fuego lento</li>
    <li>Servir y disfrutar</li>
</ol>

<!-- Lista No Ordenada -->
<h4>Ingredientes:</h4>
<ul>
    <li>Harina</li>
    <li>Azúcar</li>
    <li>Huevos</li>
    <li>Leche</li>
</ul>
